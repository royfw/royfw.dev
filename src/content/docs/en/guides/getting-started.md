Welcome. This is royfw.dev — my personal technical writing and notes space.

## Sections

Four sections, each with its own role:

- **Guides** — Step-by-step walkthroughs with full context
- **Docs** — Formal technical docs: config baselines, SOPs, system design
- **Notes** — Short captures, one tip per note
- **Projects** — What I'm building or have built

## Why two languages

Primary language is Traditional Chinese. Some posts have English versions. Switch via the language picker (top right).

:::tip[Missing content]
The English version isn't a full translation. If a post is Chinese-only, the English route falls back to the Chinese content.
:::

## Contributing

This site is open source on GitHub. Found a typo or want to add something? Hit "Edit this page" on the right side of any doc and send a PR.
